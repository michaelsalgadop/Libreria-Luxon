Librería Luxon

Util para todo tipo de manejo de fechas y horas

¿Por qué usar Luxon?

Luxon es muy útil cuando necesitas trabajar con fechas y horas de manera eficiente, especialmente si tu aplicación maneja varias Zonas horarias o requiere una manipulación compleja de fechas. Es una librería moderna, con un enfoque más limpio y menos pesado que Moment.js, y se adapta mejor a los estándares actuales de JavaScript.

Formatear fechas

Luxon ofrece métodos poderosos para mostrar fechas en diversos formatos.

Formateo estándar:

console.log(now.toISO()); // ISO 8601console.log(now.toLocaleString(DateTime.DATE_MED)); // 12 de dic. de 2024

Operaciones con Fechas

Luxon permite realizar cálculos fácilmente.

Agregar o quitar tiempo:

const sumaDias = now.plus({ days: 5 });console.log(sumaDias.toString());
const restaHoras = now.minus({ hours: 3 });console.log(restaHoras.toString());
